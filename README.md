# DebugAgent

Given an error message and code snippet, traces root cause and suggests fixes.

## Installation

```bash
pip install -r requirements.txt
cp .env.example .env  # add your ANTHROPIC_API_KEY
```

## Usage

```bash
python debug_agent.py
```

## Model

`claude-sonnet-4-20250514` via the Anthropic Python SDK.
